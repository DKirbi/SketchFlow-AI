# Transformer Patterns

**Transformer Patterns** is the UX pattern library for the **Transformers team** — a living inventory of interaction behaviours, interface flows, and composition rules for the interfaces this team builds and reviews.

---

## Two separate things

This Storybook presents **two distinct artefacts** that work together but are not the same:

### 1. Transformers Patterns (core)

The **patterns** — the interaction pattern inventory and the UPL composition model — are the **core artefact of the Transformers team**. They define:

- How interfaces should behave (validation, confirmation, save feedback, destructive flows)
- How screens are structured (UPL shell, sidebar, workspace, filter query row)
- Which interaction conventions apply across all operator tools

Patterns are designed and documented here. They are independent of any specific component library. In production, these patterns are implemented using **Podium** — the high-fidelity design system.

### 2. LOFI Kit (tool)

**LOFI Kit** is a **low-fidelity design tool** — a grayscale, monospace component library used for prototyping these patterns quickly. Lo-fi prototypes are intentionally rough: no colour, square corners, monospace type. This keeps the focus on **structure and behaviour**, not visual polish.

LOFI Kit is used here in Storybook to **render live examples** of patterns. It does not have to be used alongside the patterns — it is just the fastest way to prototype and review them before production implementation in Podium.

**LOFI Kit ≠ production design system.** Production UIs are built with **Podium**.

---

## Lo-fidelity UX testing

Lo-fi renders keep pattern examples **abstract**. When an example has no colour or brand styling, reviewers focus on structure and interaction flow — not visual preferences. For the full rationale (philosophy, greyscale-first flow, accessibility notes), see **`## Lo-fidelity UX testing`** in [`docs/UX_PATTERNS.md`](../docs/UX_PATTERNS.md) — the canonical copy lives next to **Patterns** in that file.
